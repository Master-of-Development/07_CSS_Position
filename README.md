# 모드 동아리 일곱번째 과제
- 첨부된 position.png 사진과 똑같이 만든다.
- 모든 요소에는 position 속성을 적용하여 만든다.
- position 속성을 사용하여 각 요소들의 위치를 정한다.
- 요소의 위치를 정할 때, margin, padding 속성은 사용하지 않는다.
- 9개 요소들을 감싸는 부모태그는 margin, padding 속성을 사용해도 된다.
- 모르면 구글에 검색해본후 질문한다.
